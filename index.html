What I've worked on

I developed a web application for Continental to manage bulk raw materials. I used Ruby on Rails and I was totally convinced. It was my first professional experience and I was able to accomplish all goals due to its productivity.

Talkdesk 2017 - 2021

In 2017, I started working at Talkdesk. I was part of the team that developed the first Developer Portal app. We used Ruby on Rails for the API and React for the frontend.

Later, I joined a small team that built Workspace:

  • Microfrontend engine.
  • Running 30+ apps built by different teams and technologies.
  • Written in vanilla JavaScript, a lot of iframes and postMessage and a bit of React.
  • Available both on web and desktop without any code change from the teams building the apps. We used Electron for that.

Shape Construction 2021 - now

Currently, I'm working at Shape Construction. We have a React PWA application backed by a Ruby on Rails API. My main focus initially was to reduce the amount of state we manage in the frontend, migrating the app from Redux to React Query. This improved the application reliability and also the development experience. We are doing more with less code.

For one year now, I shifted more into backend work and I'm working on our Rails API. Some examples of work I've done so far:

  • Migrate our Active Storage files to a different cloud provider without downtime.
  • Extend our existing comment feature to support commenting on other entities.
  • Develop new product to allow our users to create and manage weekly work plans for construction sites, with features like exporting to PDF via email, attaching photos, linking to other entities in the system.
